# VulnForge Pro v3.0

An automated web vulnerability assessment engine built in Python for security auditing.

> **Disclaimer:** This tool is intended solely for authorized penetration testing, educational research, and security auditing with explicit permission.

## Features
* **Reconnaissance & Info Leakage:** Identifies missing security headers, exposed secrets, and sensitive endpoints.
* **Vulnerability Analysis:** Automated logic for SQL Injection, NoSQL Injection, and Command Injection testing.
* **Evasion & Stealth Mechanics:** Configurable request throttling, User-Agent rotation, and header randomization.

## Installation

```bash
git clone [https://github.com/Ghassen-Boubaker/VulnForge.git](https://github.com/YOUR_USERNAME/VulnForge.git)
cd VulnForge
pip install -r requirements.txt
Quick Start
python3 vulnforge.py -u [https://target-example.com](https://target-example.com)
