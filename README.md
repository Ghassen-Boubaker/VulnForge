# VulnForge Pro v3.0

An automated web vulnerability scanning tool written in Python for security auditing and defensive research.

> **Disclaimer:** This tool is developed strictly for authorized security assessments, educational purposes, and defensive research. Do not run this tool against targets without explicit, written authorization.

## Features
- **Recon & Information Leakage:** Headers analysis, exposed secrets scanning, path discovery.
- **Vulnerability Checks:** SQLi, NoSQLi, and Command Injection test logic.
- **Stealth Engine:** User-Agent rotation, request delays, and header randomization.

## Installation

```bash
git clone [https://github.com/Ghassen-Boubaker/VulnForge.git](https://github.com/Ghassen-Boubaker/VulnForge.git)
cd VulnForge
